# Support Tiers

## Free / Developer

- Documentation only.
- GitHub issues.
- No private install guidance.

## Pro

- Email support.
- 48h target response.
- Support bundles accepted.

## Enterprise

- Dedicated Slack or Teams channel.
- 24h target response.
- Scheduled office hours.
- Upgrade planning.
- Support-bundle replay on vendor mirror.

## First 5 Enterprise Customers

White-glove handling regardless of tier. The goal is to build operational muscle memory and harden the deployment kit.
