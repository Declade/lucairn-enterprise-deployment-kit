  kubectl exec -n dsa-ai deploy/sandbox-b -- \
    wget -qO- --timeout=3 http://sandbox-a.dsa-identity.svc:8083/healthz

  kubectl exec -n dsa-identity deploy/sandbox-a -- \
    wget -qO- --timeout=3 http://sandbox-b.dsa-ai.svc:8084/healthz

Gateway keystore encryption (#17):
  GATEWAY_KEYSTORE_KEY is REQUIRED for gateway boot. The chart wires it via
  the gateway-credentials Secret. Provision per backend:
    k8s-native       — pass via: --set gateway.secrets.values.gatewayKeystoreKey="$(openssl rand -base64 32)"
    vault/aws/azure  — populate property `gatewayKeystoreKey` in the remote store
  Gateway will refuse to boot without it (fail-closed). Format: base64 of 32
  raw bytes (AES-256). Rotation requires re-keying the on-disk keystore;
  re-running install with a different value invalidates every encrypted record.
